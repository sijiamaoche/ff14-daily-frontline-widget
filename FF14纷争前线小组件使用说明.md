# FF14 每日纷争前线 — iOS 桌面小组件

一款基于 Scriptable 的 iPhone 桌面小组件，帮你一眼看到今天纷争前线开放的地图，无需打开游戏。

---

## 功能

- 显示今日纷争前线地图名称与模式
- 显示未来 3 天的地图排期
- 刷新倒计时
- 支持自定义背景图片（可调模糊度和亮度）
- 支持 Small（小号）和 Medium（中号）两种尺寸
- 适配 7.5 版本新轮换顺序（8 天一循环）

## 安装步骤

### 第一步：下载 Scriptable

前往 App Store 搜索 **Scriptable**，免费下载安装。

### 第二步：创建脚本

1. 打开 Scriptable App
2. 点击右上角 **「+」** 新建脚本
3. 将小组件代码**全部粘贴**到编辑器中
4. 点击左上角默认名称（Untitled Script），改为你喜欢的名字，如「FF14战场」
5. 点击 **Done** 保存

### 第三步：添加小组件到桌面

1. 回到 iPhone 主屏幕
2. **长按屏幕空白处**，进入编辑模式
3. 点击左上角 **「+」**
4. 搜索 **Scriptable**
5. 选择你想要的尺寸（推荐**中号**，信息更全）
6. 点击 **「添加小组件」**

### 第四步：绑定脚本

1. **长按**刚添加的小组件
2. 选择 **「编辑小组件」**
3. 在 **Script** 一栏选择你刚才命名的脚本（如「FF14战场」）
4. 退出编辑，等几秒小组件就会显示内容

## 设置背景图

在 Scriptable App 中**直接运行脚本**（点 ▶️ 按钮），会弹出设置菜单：

| 选项 | 说明 |
|------|------|
| 🖼️ 选择背景图 | 从手机相册选取一张图片作为小组件背景 |
| 🗑️ 恢复默认背景 | 删除自定义背景，恢复深色渐变 |
| 👁️ 预览小组件 | 不更改设置，仅预览当前效果 |

### 调整背景效果

打开脚本代码，找到顶部的两个参数：

```
OVERLAY_OPACITY = 0.65
```
控制遮罩亮暗程度。范围 0 到 1，数值越大背景越暗、文字越清晰。推荐 0.5 - 0.7。

```
BLUR_LEVEL = 3
```
控制背景模糊程度。0 表示不模糊，数值越大越模糊。推荐 3 - 5，最大 10。

修改后保存，运行一次脚本选择「预览小组件」即可查看效果。

## 校准地图

如果小组件显示的地图和当天游戏内的不一致，打开脚本代码，找到顶部：

```
const KNOWN_DATE = "2026-05-08";
const KNOWN_MAP = 3;
```

修改方法：
1. 白天打开游戏，看当前是什么地图
2. 在上方轮换表中找到这张地图对应的「第几天」（1-8）
3. 将 `KNOWN_DATE` 改为**今天的日期**（格式：年-月-日）
4. 将 `KNOWN_MAP` 改为对应的数字（1-8）
5. 保存即可

例如：5 月 8 日白天登录游戏，看到纷争前线是昂萨哈凯尔（竞争战），查表是第 3 天，就填 `"2026-05-08"` 和 `3`。

## 常见问题

**Q：小组件显示空白或报错？**

确认代码完整粘贴、没有遗漏。可以在 Scriptable 中点 ▶️ 运行一次，如果有错误会显示在底部。

**Q：小组件不刷新？**

iOS 系统会控制小组件的刷新频率，通常在每日刷新时间前后会自动更新。也可以打开 Scriptable 运行一次脚本来手动触发。

**Q：换了背景图但小组件没变？**

重新运行脚本选择「预览小组件」确认图片已保存。iOS 小组件刷新有延迟，等一会即可。如果持续不更新，可以删除小组件重新添加。

**Q：能用大号（Large）尺寸吗？**

目前只适配了小号和中号。大号会以中号布局显示。
