# FF14 每日纷争前线小组件

一款基于 Scriptable 的 iPhone 桌面小组件，帮你一眼看到今天纷争前线开放的地图，无需打开游戏。

同时提供网页版，安卓用户也可以使用。

## 预览

- 显示今日纷争前线地图名称与模式
- 显示未来排期
- 刷新倒计时
- 支持自定义背景图（可调模糊度和亮度）
- 支持 Small / Medium 两种尺寸

## iOS 用户（Scriptable 小组件）

### 安装步骤

1. App Store 下载 **Scriptable**（免费）
2. 打开 Scriptable → 点右上角 **+** → 把 `ff14战场小组件.js` 的代码粘贴进去 → 改名保存
3. 主屏幕长按 → 添加小组件 → 选 **Scriptable** → 选尺寸（推荐中号）
4. 长按小组件 → 编辑小组件 → Script 选择你的脚本

### 设置背景图

在 Scriptable 里直接运行脚本，弹出菜单：

- **选择背景图** — 从相册选一张图
- **恢复默认背景** — 回到深色渐变
- **预览小组件** — 不改设置，只看效果

代码顶部可调两个参数：

- `OVERLAY_OPACITY`：遮罩亮暗（0-1，越大越暗，默认 0.65）
- `BLUR_LEVEL`：模糊程度（0-10，越大越模糊，默认 3）

### 校准地图

如果显示的地图和游戏内不一致，打开代码，修改顶部：

```
const KNOWN_DATE = "2026-05-08";
const KNOWN_MAP = 3;
```

白天打开游戏看当前是什么地图，在上方轮换表中找到对应的「第几天」（1-8），把日期改为今天、数字改为对应序号，保存即可。
