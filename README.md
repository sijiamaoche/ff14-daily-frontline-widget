# FF14 每日纷争前线小组件

一款基于 Scriptable 的 iPhone 桌面小组件，帮你一眼看到今天纷争前线开放的地图，无需打开游戏。

同时提供网页版，安卓用户也可以使用。

## 功能

- 显示今日纷争前线地图名称与模式
- 显示未来排期
- 刷新倒计时
- 支持自定义背景图（可调模糊度和亮度）
- 支持 Small / Medium 两种尺寸

## 轮换顺序（7.5 版）

每日北京时间 23:00 刷新，8 天一循环：

| 第几天 | 地图 | 模式 |
|--------|------|------|
| 1 | ⚔️ 尘封秘岩 | 争夺战 |
| 2 | ❄️ 荣誉野 | 碎冰战 |
| 3 | 🏴 昂萨哈凯尔 | 竞争战 |
| 4 | 🐉 沃刻其特 | 演习战 |
| 5 | ⚔️ 尘封秘岩 | 争夺战 |
| 6 | 🪨 周边遗迹群 | 阵地战 |
| 7 | 🏴 昂萨哈凯尔 | 竞争战 |
| 8 | 🐉 沃刻其特 | 演习战 |

> ⚠️ 注意：尘封秘岩、昂萨哈凯尔、沃刻其特各出现了两次（第1和5、第3和7、第4和8），校准时需要区分是哪一次，详见下方校准说明。

## iOS 用户（Scriptable 小组件）

### 第一步：下载 Scriptable

前往 App Store 搜索 **Scriptable**，免费下载安装。

### 第二步：创建脚本

1. 打开 **Scriptable** App
2. 点击右上角 **「+」** 新建脚本
3. 把本仓库中 `ff14战场小组件.js` 的代码 **全部复制**，粘贴到编辑器中
4. 点击左上角的默认名称 `Untitled Script`，改一个你喜欢的名字，比如「FF14战场」
5. 点击右上角 **Done** 保存
6. 可以先点底部的 **▶️ 播放按钮** 试运行，弹出小组件预览说明代码没问题

### 第三步：添加小组件到主屏幕

1. 回到 iPhone 主屏幕
2. **长按屏幕空白处**，进入抖动编辑模式
3. 点击左上角的 **「+」** 按钮
4. 在搜索框中搜索 **Scriptable**
5. 选择你想要的尺寸（推荐 **中号 Medium**，信息更全；小号也够用）
6. 点击 **「添加小组件」**

### 第四步：绑定脚本

1. 添加完后小组件会显示空白或默认内容
2. **长按** 这个小组件
3. 选择 **「编辑小组件」**
4. 在 **Script** 那一行，选择你刚才保存的脚本名称（比如「FF14战场」）
5. 点击空白处退出编辑
6. 等几秒小组件就会刷新，显示今天的地图了

### 设置背景图（可选）

在 Scriptable App 里 **直接点击运行脚本**（点 ▶️），会弹出一个设置菜单：

| 选项 | 说明 |
|------|------|
| 🖼️ 选择背景图 | 从手机相册选取一张图片作为小组件背景 |
| 🗑️ 恢复默认背景 | 删除自定义背景，恢复深色渐变 |
| 👁️ 预览小组件 | 不更改设置，仅预览当前效果 |

代码顶部有两个参数可以微调背景效果：

- `OVERLAY_OPACITY`：遮罩亮暗，0 到 1，越大越暗，默认 0.65
- `BLUR_LEVEL`：模糊程度，0 到 10，越大越模糊，默认 3

修改后保存，运行脚本选「预览小组件」即可查看效果。

### 校准地图

如果小组件显示的地图和游戏内不一致，打开脚本代码，找到顶部：

```
const KNOWN_DATE = "2026-05-08";
const KNOWN_MAP = 7;
```

修改方法：
1. 白天打开游戏，看**今天和昨天**分别是什么地图
2. 在轮换表中找到匹配的**连续两天**，确定今天是第几天
3. 将 `KNOWN_DATE` 改为今天的日期（格式：年-月-日）
4. 将 `KNOWN_MAP` 改为对应的数字（1-8）
5. 保存即可，小组件会自动刷新

**为什么要看两天？** 因为尘封秘岩、昂萨哈凯尔、沃刻其特各出现了两次，只看一天分不清是哪一次。通过连续两天就能唯一确定位置：

| 昨天 → 今天 | 今天是第几天 |
|-------------|-------------|
| 沃刻其特 → **尘封秘岩** | **1** |
| 尘封秘岩 → **荣誉野** | **2** |
| 荣誉野 → **昂萨哈凯尔** | **3** |
| 昂萨哈凯尔 → **沃刻其特** | **4** |
| 沃刻其特 → **尘封秘岩** | **5** |
| 尘封秘岩 → **周边遗迹群** | **6** |
| 周边遗迹群 → **昂萨哈凯尔** | **7** |
| 昂萨哈凯尔 → **沃刻其特** | **8** |

例如：5 月 7 日是周边遗迹群，5 月 8 日是昂萨哈凯尔，查表「周边遗迹群 → 昂萨哈凯尔」对应第 7 天，就填 `"2026-05-08"` 和 `7`。

## 常见问题

**Q：小组件显示空白或报错？**
确认代码完整粘贴，没有遗漏。可以在 Scriptable 中点 ▶️ 运行一次，如果有错误会显示在底部。

**Q：小组件不刷新？**
iOS 系统会控制小组件的刷新频率，通常在每日刷新时间前后会自动更新。也可以打开 Scriptable 运行一次脚本手动触发。

**Q：换了背景图但小组件没变？**
重新运行脚本选择「预览小组件」确认图片已保存。iOS 小组件刷新有延迟，等一会即可。如果持续不更新，可以删除小组件重新添加。
